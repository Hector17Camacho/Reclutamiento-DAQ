# Reclutamiento-DAQ
Este repositorio contiene el desarrollo de dos actividades de adquisición, procesamiento y supervisión de señales utilizando una tarjeta ESP32 DevKitC-32E. Los proyectos incluyen el diseño del circuito electrónico, la lectura de sensores mediante las entradas ADC, el procesamiento de datos y la programación de funciones de alerta o control.
